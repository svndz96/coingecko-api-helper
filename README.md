# coingecko-api-helper
- Python helper program to work with CoinGecko API (https://www.coingecko.com/en/api)
- Customizable API calls and helper functions to integrate into other scripts

# Functions
- Ping to check API server status
- Retrieve list of all cryptocurrencies
- Retrieve data for individual cryptocurrencies, including market data, historical data, etc.
- Retrieve list of exchanges and information for each, including trust score, volume, etc.
- Retrieve global cryptocurrency market data
